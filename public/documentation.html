Achievement 2 Project: myFlix

To build the server-side component of a "movies" web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to register, deregister, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Respond body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about ALL movies, structured like:
{
    title: [title],
    year: [year],
    rating: [rating],
    description: [description],
    genre: {
        name: [name],
        description: [description]
    },
    director: {
        name: [name],
        bio: [bio],
        birth: [birth]
    },
    imageURL: [imageURL],
    featured: true/false
}
Return data about a single movie, by title /movies/[title] GET None A JSON object holding data about a single movie, structured like:
{
    title: [title],
    year: [year],
    rating: [rating],
    description: [description],
    genre: {
        name: [name],
        description: [description]
    },
    director: {
        name: [name],
        bio: [bio],
        birth: [birth]
    },
    imageURL: [imageURL],
    featured: true/false
}
Return data about a genre, by name /movies/genre/[name] GET None A JSON object holding data about the genre category and a description, structure like:
{
    genre: {
        name: [name],
        description: [description]
    }
}
Return data about a director, by name /movies/director/[name] GET None A JSON object holding data about director - name, bio, and birthdate, structured like:
{
    director: {
        name: [name],
        bio: [bio],
        birth: [birth]
    }
}
Allow users to register /users POST A JSON object holding data about the new user to register, structured like:
{
    "name": "name",
    "favoriteMovies": ["movie"]
} 		A JSON object holding data about the new user registered, structured like:
{
    name: name,
    favoriteMovies: []
    id: [id]
}
Allow users to update their user name /users/[id] PUT A JSON object holding data about the user's name to be updated, structured like:
{
    "name": "name",
} 		A JSON object holding data about the user's name updated, structured like:
{
    id: [id],
    name: [name],
    favoriteMovies: []
}
Allow users to add a movie to their list of favorites /users/id/[favoriteMovies] POST None A text message indicating the user's favoriteMovies title has been updated, structured like:
favoriteMovies has been added to user x
Allow users to remove a movie from their list /users/id/[favoriteMovies] DELETE None A text message indicating the user's favoriteMovies title has been removed, structured like:
favoriteMovies has been removed from user x
Allow existing users to deregister /users/[id] DELETE None A text message indicating the user has been deregistered, structured like:
User [id], [name], has been deregistered
